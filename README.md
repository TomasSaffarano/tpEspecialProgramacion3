⚙️ Complejidad Computacional

| Método       | Complejidad                |
| ------------ | -------------------------- |
| Constructor  | O(C + P)                   |
| Servicio 1   | O(1)                       |
| Servicio 2   | O(P)                       |
| Servicio 3   | O(P)                       |
| Backtracking | O((C+1)^P)                 |
| Greedy       | O(P log P + C log C + C·P) |


Backtracking:
- Garantiza solución óptima.
- Explora todas las combinaciones posibles.
- Alto costo computacional.

Greedy:
- Mucho más rápido.
- No garantiza solución óptima.
- Produce una solución razonable en poco tiempo.
